Benutzung des Auswertungsprogramms

Um eine neue Abrechnung zu beginnen, muss zunächst auf den Knopf "NEU" geklickt werden. Dadurch werden alle alten Daten gelöscht. Kassendaten können mit einem Klick auf den Knopf "ÖFFNEN" eingelesen werden. Die Dateien mit den Kassendaten haben die Endung ".flohmarkt" und können entweder einzeln oder gleichzeitig ausgewählt und damit eingelesen werden. Das Programm merkt sich, welche Dateien schon eingelesen wurden, um mehrfaches Einlesen zu vermeiden.

Die Tabellen mit der Auswertung können direkt ausgedruckt werden. Möchte man den Ausdruck aufwendiger gestalten, z.B. Briefumschläge für jeden Anbieter ausdrucken, dann können die Daten in eine ".csv"-Datei exportiert werden, die sich dann mit einem Textverarbeitungsprogramm weiter nutzen lässt.

Kasse

Einrichten der Kasse

Das Programm ist optimiert für den Start von einem USB-Stick, da alle Konfigurationen und die Datendatei dann dort gleich gespeichert werden. Das Programm funktioniert aber auch ohne Einschränkung von einer Festplatte oder anderem Speichermedium aus.

Nach dem ersten Start des Kassenprogramm muss das Kassenprogramm eingerichtet werden. Dazu wird auf den Knopf "NEU" gedrückt. Es wird empfohlen, eine automatische Einrichtung zu wählen. Dann merkt sich das Programm den Kassennamen und kann das nächste Mal ohne erneute Einrichtung wieder verwendet werden. Das Programm legt, wenn es mehr als 24 Stunden nicht geöffnet wurde, von alleine eine neue Datei an, so dass auch zwischen den Flohmärkten unterschieden werden kann.

Es wird dennoch empfohlen, alte .flohmarkt-Dateien vor einem neuen Flohmarkt zu löschen, um Verwechslungen zu vermeiden.

Benutzung

Nach der Einrichtung kann sofort mit dem Kassieren beginnen.

Das gesamte Programm kann mit der Nummerntastatur bedient werden. Der Cursor geht nach jeder Eingabe in das Feld “Anbieternummer“. Jede Eingabe kann mit der Eingabetaste bestätigt werden, woraufhin der Cursor in das nächste Feld “Preis“ springt. Das Programm warnt, wenn Werte eingegeben werden, die falsch sein könnten, so z.B. zu hohe Preise, krumme Preise und Anbieternummern, die nicht vergeben werden. Dennoch lassen sich diese Eingaben nach einer Bestätigung eingeben. Eine falsche Eingabe kann einfach mit einem Mausklick auf den Löschknopf der Zeile gelöscht werden.

Ein Kassenvorgang wird mit der Eingabe des Pluszeichens “+“ abgeschlossen.

Soll nach dem Kassiervorgang noch ein Posten gelöscht werden, so kann mit einem Klick auf den Bearbeitenknopf eine Übersicht aller Kassiervorgänge aufgerufen werden. Der zu löschende Vorgang wird angeklickt und dann das Fenster geschlossen. Wie bei jedem Kassiervorgang ist nun ein negativer Betrag eingegeben, der mit der Eingabetaste bestätigt werden kann.

Das Kassenprogramm kann ohne Probleme zwischenzeitlich auch geschlossen werden. Wird es innerhalb von 24 Stunden wieder gestartet, werden die Kassenvorgänge in derselben Datei wieder gespeichert. Sonst wird eine neue Datei angelegt.
